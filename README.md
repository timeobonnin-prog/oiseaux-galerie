# Galerie secrète d’oiseaux 🐦

Un site en un seul fichier (HTML/CSS/JS) qui affiche mes photos d’oiseaux avec un filigrane automatique pour éviter le vol.

## Fonctionnalités
- Accès protégé par un code
- Ajout de photos depuis mon ordinateur
- Filigrane automatique anti‑copie
- Galerie en grille responsive
- Marquage des “meilleures” photos
- Design doré / nuit

## Objectif
Créer une interface simple et stylée pour organiser mes photos d’oiseaux sans qu’on puisse les voler.
